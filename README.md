# COACHTECH勤怠管理アプリ

## 概要
このプロジェクトは、ユーザーの勤怠と管理を行うアプリです。一般ユーザー 又は 管理者ユーザー のマルチログイン方式です。  
一般ユーザーは、会員登録・ログイン・勤怠登録ができるほか、月毎の勤怠一覧を確認し勤怠情報の修正申請を行うことができます。  
管理者ユーザーは、日毎の勤怠、ユーザー毎の勤怠、ユーザー一覧を確認できるほか、一般ユーザーから修正申請を受けた勤怠の確認及び修正承認ができます。
